有名なライブラリ＆ゲームエンジン達

始めに行ってしまうとゲームに関係するライブラリやゲームエンジンは星のようにある。
なので今回は有名な極々一部だけを紹介していく。

ぜひ気になったものがあったら、自分の手で調べてほしい。

DirectX

windowsが標準に搭載する。画像処理などゲームに関わる処理のAPI群。(ちなみにXboxなんかにも使われている)

新たな言葉が出てきたが説明は省かせてもらう。windows上でゲームを動かす時の共通規格の集まりとだけ分かってくれれば大丈夫。
※詳しく知りたかったら各々調べてね。

windows上で動くゲームはほぼほぼコレが使われている。
ゲームエンジンやゲームライブラリなんかもコレを元に作られているのがほとんど。

初心者がゲームを作る時にコレを選択すると、5000兆%挫折すると思う。少なくとも主は挫折した。

実際に触ってみれば分かるが、ウィンドウの表示だけでかなりの行数を書くことになる。(それ自体はDirectXではなくWindowsAPIと呼ばれるものだが)

しかしほとんどのゲームの開発環境の底にはコイツが存在するので、ゲームプログラマーなどを目指すのであれば習得しておきたい。

「もっと簡単にコレを使うことは出来ないの？」

安心してほしい、これらをとても簡単に使うことが出来るライブラリがある。
(といっても"比較的"にだが)

主要言語: C , C++ など

DXライブラリ

先程紹介した"DirectX"をライブラリ化したのがこの"DXライブラリ"だ。

DirectX単体では長々と書かなければ行けなかったものを、DXライブラリでは数行程度で済んだりする。その分DirectXほど自由度は高くないが、まず個人開発でそこまでのものを作ることはないと思いたい。(それは素直にゲームエンジンを使おう)

しっかりとコードを書いてゲームを作りたい。
どのような処理で動いているのかを知っておきたい
等の考えがある場合、大いに役に立つライブラリだ。

これを足場にDirectXに移るのもとてもアリだと思う。

最近PS4やPSvita環境で動くDXライブラリが発表された。

主要言語: C++ ※ただしCの知識だけでも十分に使用可能。

OpenGL

DirectXと同じように2D、3D処理に特化したAPI群だ。

比較的に簡単なものなのでDirectXほど難しくはない。

DirectXとの一番の違いは、環境を選ばずに動くという点である。
これのお陰で非常に優れた移植性を持つ。
※他の環境に移しても動くということ。

コレはゲームよりも、3DCGソフト等の3D処理が必要なソフトウェアに使われることが多く、使用用途はとても幅広い。

主要言語: C , C++ , GLSL(OpenGL Shading Language) ※OpenGLの専用言語

cocos2d-x

先程まではライブラリ関連の紹介をしてきたが、ここからはゲームエンジンを紹介する。

これは名前の通り、2Dゲーム用のゲームエンジンだ。
オープンソースで開発されているので金銭面に悩むことはないし、慣れてきたら自分でソースコードを読んで自分なりにカスタマイズすることも可能だ。

慣れ親しんだGUIでも、硬派にCUIで開発することもできる。もちろん組み合わせることもできる。

コレ一つでandroid、iOS、windows、webアプリなどのアプリを作ることができるし、
2Dに特化し、C++をベースに作られているため、豊富な機能を備えているにも関わらず、大変軽量なエンジンとなっている。

しかし3Dエンジンによくある自動的に当たり判定を設定したり、物理エンジン使おうとすると、それなりの知識が必要になってくる点が初心者にはとっつきにくいかもしれない。

主要言語: C++ , JavaScript

Unity

ここ数年で一気にシェアを伸ばしたゲームエンジンの"Unity"だ。

シェアを伸ばしたのにはもちろん理由がある。

結論を言ってしまうとプログラムを一切書かずにゲームを作ることが可能だ。

このゲームエンジンはとにかくとっつきやすいゲームエンジンだ。
一応3Dゲームエンジンだが少し切り替えボタンを押すだけで2Dゲームエンジンにすることもできる。

基本的にすべて用意されたGUIの中で操作する。
ステージを作るのも、アニメーションを作るのも、オブジェクトに対してプログラムを適用するのも、すべてGUIで事足りる。

一番の使用するメリットがあるのは、アセットストアと呼ばれる、ゲーム内で使う素材を購入することができるストアが公式に用意されているところだろう。
おまけに買ったらすぐにUnityのGUI上に反映されるという親切っぷり。
3Dモデルから背景素材までなんでも揃ってる上に、無料の素材もかなりの数がある。

Unity上でゲームに関する全てが揃ってしまうといっても過言ではない。

主要言語: C# , JavaScript ※独自要素が多すぎてUnityScriptと呼ばれている。

UnrealEngine

超絶美麗グラフィック3Dゲームは大抵これで作られてる思え。

はい。超綺麗な3Dゲームを作ってみたいと思ったあなた、コレを使いましょう。

UnrealEngine自体はだいぶ前からあるエンジンだが、ここ数年でUnityが現れ、かなりのシェアを持っていかれた...が
なんとアセットストアがあります。というか出来ました。
超絶キレイなグラフィック素材が買えます。3Dモデルも変えます。なんならアニメーションやモーションも売ってます。

最近はUnrealEngineに触ったことない人向けにチュートリアルなども充実している。

そして一番の特徴は"ブループリント"という機能だ。
UnrealEngineはゲームに関する関数群の数、種類がやばい。どんな状況でそれ使うの？って関数も結構ある。
それらを視覚的に組み合わせてプログラムを組むことが出来る。
この機能は一度体験してもらいたい、本当に。
プログラムについて勉強してこなかった人でもそこそこのゲームが作れるくらいには優れている。

欠点としては重い。これで作ったゲームも、このエンジン自体も。
俗に言うゲーミングPCって代物じゃないとまともに動かない。

主要言語: C++ , BluePrint ※これすごい。(語彙力)
